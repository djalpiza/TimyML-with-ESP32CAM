Detailed Description

This function resizes all jpeg images in a specified input directory to a specified size and saves them in a specified output directory using nearest neighbor resampling method.

Description

This function rescales all images in the input_dir and stores the scaled images in the output_dir. Parameters:

  • input_dir (str): The directory where the original images are stored
  • output_dir (str): The directory where the scaled images will be stored Returns: None
