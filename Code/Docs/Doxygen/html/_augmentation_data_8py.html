Detailed Description

This script demonstrates the use of various libraries for working with images.

Description

This function performs data augmentation on the images present in the input_dir. It applies various image transformation such as width_shift, height_shift, brightness_range and rotation_range on the images and saves the augmented images in the same input_dir with a different name. Args: input_dir: str: The path of the directory containing the images max_images: int: The maximum number of augmented images to be generated per original image Returns: None
